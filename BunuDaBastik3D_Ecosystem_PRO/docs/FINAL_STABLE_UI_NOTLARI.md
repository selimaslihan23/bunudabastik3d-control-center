# BunuDaBastık3D Control Center FINAL STABLE

Bu sürüm Safe Mode çekirdeği üzerine kuruldu. Splash/login kaynaklı takılmalar kaldırıldı; uygulama direkt ana panele açılır.

## Eklenenler

- Operatör seçimi: Selim / Kemal
- Ayarlar > Arayüz, operatör ve animasyon ayarları
- Tema varyasyonları:
  - Nebula Mor
  - Cyber Gold
  - Matrix Yeşil
  - Buz Mavisi
  - Sade Pro
- Arka plan/ambiyans varyasyonları:
  - Statik
  - Aurora Pulse
  - Neon Grid
  - Yıldız Tozu
  - Enerji Çizgileri
- Buton stili varyasyonları:
  - Glow Neon
  - Solid Pro
  - Neon Çerçeve
  - Minimal
- Animasyon seviyesi:
  - Kapalı
  - Hafif
  - Canlı

## Önerilen kullanım

Günlük kullanım için `run_windows.bat` dosyasını çalıştırın. EXE üretmek için `build_windows_FINAL_STABLE.bat` kullanın.

## En sağlam EXE yolu

ZIP'i `C:\BDB\` içine çıkarın. Türkçe karakterli klasör yolları PyInstaller tarafında sorun çıkarabildiği için kısa ve düz klasör yolu önerilir.

## Log dosyaları

- `logs\run_final_stable_console.log`
- `%LOCALAPPDATA%\BunuDaBastik3D\logs\app_debug.log`
