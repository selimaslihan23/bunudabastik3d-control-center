# Odoo Entegrasyonu

Varsayılan adres:

```text
https://bunudabastik3d.odoo.com
```

Uygulama Odoo tarafında şu kayıtları oluşturmak için hazırlanmıştır:

- Tekliflerden CRM fırsatı
- Siparişlerden CRM fırsatı
- Katalog ürünlerinden ürün kartı

## Gerekli alanlar

- Odoo URL
- Database adı
- Kullanıcı e-posta
- API Key

## Kullanım

1. Entegrasyonlar sekmesinde ayarları kaydet.
2. Odoo Test butonuyla bağlantıyı doğrula.
3. Senkron sekmesinde bekleyen kayıtları Odoo'ya gönder.

Not: Odoo modülleri kurulu değilse `crm.lead` veya `product.template` oluşturma hatası alınabilir.
