# Katalog SmartPrice Düzeltmesi

Bu sürümde MakerWorld / Bambu Studio değerleri varsayılan olarak **Toplam** kabul edilir.

Doğru kullanım:

- MakerWorld veya Bambu Studio size plaka/toplam değer veriyorsa:
  - Gram tipi: Toplam
  - Süre tipi: Toplam

- Tek parça için gram/süre giriyorsanız:
  - Gram tipi: Parça başı
  - Süre tipi: Parça başı

Uygulama parça başı modda değerler aşırı büyürse uyarı verir.
