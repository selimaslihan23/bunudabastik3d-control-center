# Katalog ID Düzenleme Modu

Bu sürümde Katalog ekranına güvenli ID düzenleme modu eklendi.

## Ne işe yarar?
- Seçili katalog ürününün uygulama içi `Kayıt ID` değerini değiştirebilirsin.
- Seçili katalog ürününün `Model ID` değerini elle düzeltebilirsin.
- MakerWorld linkinden ID yanlış çıkarsa veya link değiştiyse Model ID elle yazılabilir.

## Kullanım
1. Katalog tablosundan ürünü seç.
2. `Seçileni Forma Al` butonuna bas.
3. `ID düzenleme modu` kutusunu işaretle.
4. `Kayıt ID` veya `Model ID` alanını değiştir.
5. `ID / Model ID Güncelle` butonuna bas.

## Güvenlik
- ID düzenleme modu kapalıyken kayıt ID değiştirilemez.
- Girilen ID pozitif tam sayı olmalıdır.
- Aynı ID başka üründe varsa işlem yapılmaz.
- ID değişimi öncesinde onay sorulur.

Not: Normal kullanımda Kayıt ID otomatik kalmalıdır. Sadece katalog düzenini elle toparlamak veya dış sistemlerle ID eşleştirmek gerektiğinde kullan.
