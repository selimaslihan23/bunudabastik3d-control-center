# BunuDaBastık3D Control Center v3.0 Premium UI

Bu sürümde arayüz tarafına marka odaklı premium katman eklendi.

## Eklenenler

- Logo destekli splash screen
- Yerel operatör giriş ekranı
- Mor/siyah premium tema
- Sol menüye gerçek BunuDaBastık3D logo görseli
- Windows pencere ikonu ve PyInstaller ikon ayarı
- Dashboard hızlı işlem barı
- Klavye kısayolları:
  - F5: tüm verileri yenile
  - Ctrl+N: Gelen Kutusu
  - Ctrl+E: e-posta çek
  - Ctrl+B: Bridge kontrol
  - Ctrl+K: Katalog
- Premium dashboard önizleme görselleri

## Güvenlik notu

Giriş ekranı yerel operatör karşılama ekranıdır. API anahtarı veya gerçek parola doğrulaması değildir. Gerçek yetkilendirme istenirse sonraki sürümde kullanıcı tablosu, hashlenmiş parola ve rol bazlı yetki sistemi eklenmelidir.
